## Rodando a Aplicação Localmente

Execute a sequencia de comandos para executar a aplicação:

```bash
git clone https://github.com/Mayxx-x/facilita_limpezas
```

```bash
cd facilita_limpezas
```

```bash
npm install
````

```bash
npm run dev
```
